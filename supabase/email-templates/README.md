# Email templates (Amby Luxe style)

Use these in **Supabase Dashboard → Authentication → Email Templates**.

1. Open each template type (Confirm signup, Magic Link, Reset Password).
2. Paste the matching HTML from the `.html` files here.
3. Save.

Design: dark background (`#0a0c10`), gold accent (`#d4af37`), serif headings – matches the site UI.

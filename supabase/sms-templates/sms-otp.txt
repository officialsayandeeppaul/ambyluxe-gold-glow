# Amby Luxe SMS OTP Template

Use in **Supabase Dashboard → Authentication → Providers → Phone → Message** (or SMS template field).

Supabase uses `{{ .Token }}` for the 6-digit OTP. Some systems use `{{ .Code }}` — swap if needed.

---

## Recommended (elegant, on-brand)

Your verification code for Amby Luxe is {{ .Token }}. Valid for 60 seconds. — sayandeep.store

---

## Alternatives

**Short & clean:**
Amby Luxe: {{ .Token }} — Your sign-in code. Expires in 60 sec.

**Minimal:**
{{ .Token }} — Amby Luxe verification. Expires soon.

**Warm & personal:**
Welcome to Amby Luxe. Use {{ .Token }} to sign in. This code expires in 60 seconds.
